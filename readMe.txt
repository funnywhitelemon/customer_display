Инструкция по подключению:
    1. Ставим в базу расширение из папки cfe
    2. Устанавливаем платформу Node.js версию 16 и выше. Cкачать тут https://nodejs.org/ 
    3. Открываем корневую папку проекта в консоли и пишем - npm install
    5. Запускаем сервер командой npm run serve (в консоли при этом открыта корневая папка проекта)
    6. Запускаем файл index.html из папки front в любом современном браузере и выносим его на второй монитор
    7. Радуемся !! ))
